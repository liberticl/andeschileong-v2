Proyecto konta2r

Sistema de deteccion de transito en tiempo real que utiliza vision artificial para identificar y contar diferentes tipos de usuarios en la via publica.

Modelo: YOLO26n con ONNX Runtime Web.
Backend: WebGPU (preferido) o WASM con fallback automatico.
Fuente: URL externa o archivo local.

Desarrollado originalmente por Ariel Lopez.
Version YOLO26n por Francisco Vergara de Andes Chile ONG.
