Proyecto konta2r

Este es un sistema de detección de tránsito en tiempo real que utiliza visión artificial para identificar y contar diferentes tipos de usuarios y elementos móviles en la vía pública.

El sistema puede detectar:

  • Peatones y ciclistas
  • Patinetas, autos, motos, buses y camiones
  • Mascotas (perros y gatos)

Desarrollado por Ariel López utilizando TensorFlow y el modelo COCO-SSD para detección de objetos en tiempo real. Para más información puede ver el artículo del proyecto.

Esta versión es para uso educativo y social sin fines de lucro. Si desea utilizarlo para estudios o proyectos con fines comerciales contacte al autor.
