Proyecto konta2r

Sistema de deteccion de transito en tiempo real que utiliza vision artificial para identificar y contar diferentes tipos de usuarios y elementos moviles en la via publica.

Modelo: YOLO26n (mAP 40.9, 2.4M parametros) con ONNX Runtime Web.

Backend: WebGPU (preferido) o WASM con fallback automatico.

Resolucion adaptativa: 320px (dispositivos con ≤2GB RAM) o 640px (dispositivos con más RAM).

El sistema puede detectar:

  • Peatones y ciclistas
  • Patinetas, autos, motos, buses y camiones
  • Mascotas (perros y gatos)

Desarrollado originalmente por Ariel Lopez. Version YOLO26n para Andes Chile ONG.
